# Roadmap – Odoo EduCat Lazitos

## Phase 0 – Discovery
- Definir MVP (qué entra / qué no)
- Roles y permisos: Admin / Docente / Padre

## Phase 1 – Infra
- VM o local Docker
- Docker Compose listo
- (Opcional) Nginx + SSL

## Phase 2 – Odoo + EduCat
- Odoo base funcional
- Instalar EduCat Core + módulos académicos

## Phase 3 – Configuración funcional
- Cursos / salas
- Alumnos / tutores
- Asistencia
- Pagos / cuotas

## Phase 4 – UX & adopción
- Simplificar menús
- Portal padres

## Phase 5 – Mejoras
- Backups automáticos + restore probado
- Emails automáticos
- Reportes PDF
- Monitoreo / logs
